Projeto REA-Stat - Recursos Educacionais Abertos de Estatística

O propósito desse projeto é manter uma biblioteca virtual abrangente e dinâmica de recursos educacionais para o ensino de Estatística en nível universitário, includindo apostilas, livros, tabelas, slides, exercícios, etc.

Biblioteca virtual

  • Notas de Aula: Probabilidade I (por Augusto Teixeira). Disponível no GitHub e Overleaf
  • Tabelas Estatísticas. Disponível no GitHub e Overleaf
  • Testes de Hipótese (por Leonardo T. Rolla). Disponível no GitHub e Overleaf
  • Livro "Probabilidade e Estatística" (EAD-UFRGS). Disponível no GitHub, pdf e em html
  • Livros de Álgebra Linear, Cálculo, Cálculo Numérico, etc. (REAMAT-UFRGS). Disponível na página do REAMAT

Critérios e objetivos

Serão incluídos nesta biblioteca recursos educacionais que cumpram três condições:

  • Seja escrito em português
  • Trate de conteúdo de Estatística ou áreas afins
  • Tanto o produto final quanto o código-fonte devem ser abertos

O primeiro objetivo deste projeto é servir como uma base comum e motivadora para que os vários colegas ao redor do mundo compartilhem seus recursos didáticos de forma aberta e sistemática.

O segundo objetivo é que esta biblioteca virtual seja dinâmica e continue crescendo ao longo dos anos. Para isso, o material listado nesta biblioteca não será sequencial nem estruturado, e tampouco terá que passar necessariamente por uma rigorosa avaliação de sua adequação a algum propósito específico. Assim sendo, o público-alvo não são os alunos, mas sim os professores de Estatística em nível universitário, que deverão avaliar o material por conta própria antes de utilizá-lo em sua prática docente.

O que significa ser aberto?

Este é o nosso critério para dizer que um recurso educacional em estatística é verdadeiramente aberto:

  • O produto final deve estar publicado sob alguma das licenças CC-0, CC-BY ou CC-BY-SA.
  • O código-fonte do texto também deve estar publicado sob uma dessas licenças.
  • O código-fonte do texto deve estar em formato aberto (LaTeX, Rexams, Rbookdown, etc).
    • Por razões de portabilidade, acessibilidade, e reprodutibilidade, não incluiremos material escrito em MS-Office, iWork, OpenOffice e similares, ainda que estejam disponibilizados de forma aberta.
  • Os programs usados na geração de dados e figuras que façam parte integral do texto devem estar publicados com licença MIT, LGPL, GPLv2 ou GPLv3, ou incorporados ao código-fonte do texto.
    • Tais programas não podem depender de software proprietário para rodar (como Excel, Minitab ou SPSS), exceto quando a finalidade do material for o ensino do uso do software em questão.
    • Este requisito não se aplica a dados e figuras que sejam meramente ilustrativos e possam ser omitidos do texto sem prejuízo de conteúdo.

Eventualmente poderemos criar uma seção separada com materiais parcialmente abertos, que são aqueles sem código fonte adequado ou cuja licença seja compatível com CC-BY-ND.

Material publicado sob a licença CC-BY-NC não é considerado como um recurso aberto.
