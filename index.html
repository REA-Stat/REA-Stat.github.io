Projeto REA-Stat - Recursos Educacionais Abertos de Estatística

O propósito desse projeto é manter uma biblioteca virtual abrangente e dinâmica de recursos educacionais para o ensino de Estatística en nível universitário, includindo apostilas, livros, tabelas, slides, exercícios, etc.

Biblioteca virtual

  • Tabelas Estatísticas. Disponível no GitHub e Overleaf
  • Testes de Hipótese (por Leonardo T. Rolla). Disponível no GitHub e Overleaf

Critérios e objetivos

O primeiro objetivo é servir como uma base comum e motivadora para que os vários colegas ao redor do mundo compartilhem seus recursos didáticos de forma aberta e sistemática.

O segundo objetivo é que esta biblioteca virtual seja dinâmica. Para isso, o material listado nesta biblioteca não será sequencial nem estruturado, e tampouco terá que passar necessariamente por uma rigorosa avaliação de sua adequação a algum propósito específico. Assim sendo, o público-alvo não são os alunos, mas sim os professores de Estatística em nível universitário, que deverão avaliar o material por conta própria antes de utilizá-lo em sua prática docente.

Serão incluídos nesta biblioteca recursos educacionais que cumpram três condições:

  • Seja escrito em português
  • Trate de contúdo de Estatística ou áreas afins
  • Tanto o produto final quanto o código-fonte devem ser abertos

Mas o que significa ser aberto? Este é o nosso critério para dizer que um recurso educacional em estatística é verdadeiramente aberto:

  • O produto final deve estar publicado sob alguma das licenças CC-0, CC-BY ou CC-BY-SA
  • O código-fonte do texto também deve estar publicado sob uma dessas licenças
  • A parte de software usada na geração de dados e figuras deve estar publicado com licença compatível com GPLv2 ou GPLv3 (incluindo MIT e LGPL)
  • O código-fonte do texto deve estar em formato aberto (tex, rexams, rbookdown, etc) e também publicado sob uma dessas licenças
  • As tabelas e figuras não devem geradas usando software proprietário, como Excel, Minitab ou SPSS (exceto quando a finalidade do material for o ensino do uso do software em questão)

Por razões de portabilidade, acessibilidade, e reprodutibilidade, não publicamos material escrito em MS-Office, iWork, OpenOffice e similares, ainda que publicados de forma aberta.

Eventualmente poderemos criar uma seção separada com materiais parcialmente abertos, que são aqueles sem código fonte adequado ou cuja licença seja compatível com CC-BY-ND.

Material publicado sob a licença CC-BY-NC não é considerado como parcialmente aberto.
