Projeto REA-Stat - Recursos Educacionais Abertos de Estatística

O propósito desse projeto é manter uma biblioteca virtual abrangente e dinâmica de recursos educacionais para o ensino de Estatística em nível universitário, includindo apostilas, livros, tabelas, slides, exercícios, etc.

Biblioteca virtual

  • GitHub, Overleaf - Banco de Exercícios de Estatística (construção coletiva).
  • GitHub, Overleaf - Notas de Aula: Probabilidade I (por Augusto Teixeira).
  • GitHub, Overleaf - Tabelas Estatísticas.
  • GitHub, Overleaf - Testes de Hipótese (por Leonardo T. Rolla).
  • GitHub, pdf, html - Livro "Probabilidade e Estatística" (EAD-UFRGS).
  • página - Livros de Álgebra Linear, Cálculo, Cálculo Numérico, etc. (REAMAT-UFRGS).

Critérios e objetivos

Serão incluídos nesta biblioteca recursos educacionais que cumpram três condições:

  • Seja escrito em português
  • Trate de conteúdo de Estatística ou áreas afins
  • O recurso deve estar disponível de forma totalmente aberta

O primeiro objetivo deste projeto é servir como uma base comum e motivadora para que os vários colegas ao redor do mundo compartilhem seus recursos didáticos de forma aberta e sistemática.

O segundo objetivo é que esta biblioteca virtual seja dinâmica e continue crescendo ao longo dos anos. Para isso, o material listado nesta biblioteca não será sequencial nem estruturado, e tampouco terá que passar necessariamente por uma rigorosa avaliação de sua adequação a algum propósito específico.
